***Date Formatter***

Простая демонстрация полиморфизма в C#, реализующая форматирование дат в различных региональных стандартах.

**Описание**

Проект представляет собой учебный пример, демонстрирующий принципы объектно-ориентированного программирования, в частности наследование и полиморфизм. Реализована иерархия классов для форматирования дат в трех различных форматах: базовом, американском и европейском.

**Особенности**

*Наследование:* Создана иерархия классов, где AmericanDate и EuropeanDate наследуются от базового класса BaseDate.

*Полиморфизм:* Используется переопределение виртуального метода GetFormat() для обеспечения разного поведения в каждом классе-наследнике.

**Разные форматы дат:**

*Базовый*: "год:ГГГГ, месяц:ММ, день:ДД"

*Американский:* "ММ.ДД.ГГГГ"

*Европейский*: "ДД.ММ.ГГГГ"

Универсальная обработка: Возможность хранить объекты разных классов в одной коллекции типа List<BaseDate> и вызывать для каждого свой вариант метода GetFormat().

**Структура проекта**

*BaseDate* — базовый класс даты с свойствами Year, Month, Day и виртуальным методом GetFormat().

*AmericanDate* — класс-наследник, переопределяющий метод GetFormat() для вывода даты в формате MM.DD.YYYY.

*EuropeanDate* — класс-наследник, переопределяющий метод GetFormat() для вывода даты в формате DD.MM.YYYY.

Program — основной класс с точкой входа, создающий экземпляры всех типов дат и демонстрирующий работу полиморфизма.
